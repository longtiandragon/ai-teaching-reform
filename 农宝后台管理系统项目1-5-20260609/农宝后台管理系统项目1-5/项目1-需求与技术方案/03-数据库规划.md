# 03-数据库规划

## 一、数据库设计原则

- 表名使用小写加下划线。
- 主键使用 `id`。
- 常用业务表保留创建时间、更新时间、备注、状态字段。
- 状态字段用于控制启用、停用、发布、下架等业务状态。
- 图片、视频和附件只保存访问地址，不直接保存文件内容。

## 六、公共字段建议

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | varchar(64) | 主键 |
| create_time | datetime | 创建时间 |
| update_time | datetime | 更新时间 |
| create_by | varchar(64) | 创建人 |
| update_by | varchar(64) | 更新人 |
| status | int / char | 状态 |
| remark | varchar(500) | 备注 |
