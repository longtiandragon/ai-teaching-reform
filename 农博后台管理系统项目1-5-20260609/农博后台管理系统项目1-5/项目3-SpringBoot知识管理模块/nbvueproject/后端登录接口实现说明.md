## 🚀 如何使用

### 方式1：使用 Spring Boot 后端（推荐）

#### 1. 确保数据库已创建并导入数据

```sql
-- 1. 创建数据库（如果没有）
CREATE DATABASE nb_database CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 2. 导入初始数据
USE nb_database;
source nbspringproduct/src/main/resources/sql/system_tables.sql;
source nbspringproduct/src/main/resources/sql/system_init_data.sql;

-- 3. 修复乱码（如果有乱码）
source nbspringproduct/src/main/resources/sql/fix_encoding.sql;
```

#### 2. 修改数据库配置（如有必要）

编辑 `nbspringproduct/src/main/resources/application.yml`：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/nb_database?useUnicode=true&characterEncoding=utf8&zeroDateTimeBehavior=convertToNull&useSSL=false&serverTimezone=GMT%2B8&allowPublicKeyRetrieval=true
    username: root
    password: 123456  # 改成你的数据库密码
```

#### 3. 启动 Spring Boot 后端

```bash
cd nbspringproduct
mvn spring-boot:run
```

或者在 IDE（IDEA/Eclipse）中直接运行 `NbspringproductApplication.java`

#### 4. 修改前端配置

编辑 `nbvueproject/src/utils/request.ts`：

```typescript
// Spring Boot 版本后端（推荐）
const baseURL = 'http://localhost:80'
const isSpringBoot = true  // Spring Boot 版本
```

#### 5. 启动前端

```bash
cd nbvueproject
npm run dev
```

#### 6. 测试登录

访问 `http://localhost:5173`（或前端显示的地址）

使用以下账号登录：
- **账号：** admin
- **密码：** 123456

## 🎯 前后端联调流程

### 完整的登录流程

```
┌─────────────────────────────────────────────────────────┐
│ 1. 用户在前端输入账号密码                                │
│    账号: admin                                           │
│    密码: 123456                                          │
└─────────────────┬───────────────────────────────────────┘
                  ↓
┌─────────────────────────────────────────────────────────┐
│ 2. 前端 Pinia 调用登录方法                               │
│    userStore.login('admin', '123456')                   │
└─────────────────┬───────────────────────────────────────┘
                  ↓
┌─────────────────────────────────────────────────────────┐
│ 3. 发送 HTTP 请求到后端                                  │
│    POST http://localhost:80/dev-api/yjnb/system/login   │
│    Body: {"username":"admin","password":"123456"}       │
└─────────────────┬───────────────────────────────────────┘
                  ↓
┌─────────────────────────────────────────────────────────┐
│ 4. 后端 AuthController 接收请求                          │
│    - 验证用户名密码                                      │
│    - 查询数据库                                          │
│    - 生成 Token                                          │
└─────────────────┬───────────────────────────────────────┘
                  ↓
┌─────────────────────────────────────────────────────────┐
│ 5. 后端返回成功响应                                      │
│    {                                                     │
│      "code": 200,                                        │
│      "data": {                                           │
│        "username": "admin",                              │
│        "token": "TOKEN_admin_xxx",                       │
│        "role": "超级管理员"                              │
│      }                                                   │
│    }                                                     │
└─────────────────┬───────────────────────────────────────┘
                  ↓
┌─────────────────────────────────────────────────────────┐
│ 6. 前端 Pinia 保存状态                                   │
│    - 保存到 Pinia State（内存）                          │
│    - 保存到 localStorage（持久化）                       │
│    - 不再使用本地降级验证 ✅                             │
└─────────────────┬───────────────────────────────────────┘
                  ↓
┌─────────────────────────────────────────────────────────┐
│ 7. 跳转到主界面                                          │
│    router.push('/main')                                 │
└─────────────────────────────────────────────────────────┘
```

## ⚠️ 注意事项

### 1. CORS 跨域配置

两个后端都已配置 CORS，允许跨域访问。

**Spring Boot：**
```java
@CrossOrigin  // 在 Controller 类上
```

**SSM：**
```java
@CrossOrigin(originPatterns = "*", allowCredentials = "true")
```

### 2. 字符编码

确保以下配置正确：

**数据库连接字符串：**
```
useUnicode=true&characterEncoding=utf8
```

**数据库字符集：**
```sql
CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci
```

### 3. 端口占用

- Spring Boot：默认 80 端口
- SSM (Tomcat)：默认 8080 端口
- Vue 前端：默认 5173 端口

确保端口没有被占用。

### 4. 数据库连接

确保 MySQL 服务已启动，且账号密码正确。
