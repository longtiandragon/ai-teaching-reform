# 02-技术方案规划

## 一、总体架构

系统采用前后端分离架构：

```text
Vue 前端管理系统
        |
        | HTTP / JSON
        |
Spring MVC / Spring Boot 后端服务
        |
        | MyBatis / MyBatis-Plus
        |
MySQL 数据库
```

## 二、技术选型

| 层级 | 技术 | 说明 |
| --- | --- | --- |
| 前端 | Vue 3 | 构建后台管理页面 |
| 前端构建 | Vite | 前端开发和打包 |
| UI 组件 | Element Plus | 表格、表单、弹窗、菜单 |
| 状态管理 | Pinia | 保存登录用户信息 |
| HTTP 请求 | Axios | 调用后端接口 |
| 项目2后端 | Spring + Spring MVC + MyBatis | SSM 基础后端 |
| 项目3-5后端 | Spring Boot 3 + MyBatis-Plus | 简化配置和 CRUD |
| 数据库 | MySQL | 保存系统和业务数据 |
| 构建工具 | Maven / npm | 后端和前端构建 |

## 三、后端分层设计

```text
controller：接收请求，返回统一结果
service：处理业务逻辑
mapper：访问数据库
entity：实体类
common：统一返回、分页参数、公共实体
config：跨域、MyBatis-Plus、文件上传配置
```

## 四、前端页面设计

```text
login.vue：登录页
Main.vue：后台主布局
Menu.vue：左侧菜单
views：业务页面
utils/request.ts：Axios 请求封装
store/user.ts：登录状态管理
router/router.ts：路由配置
```

## 五、迭代架构说明

### 项目2

使用 SSM 实现基础后端，主要体现传统 Java Web 后端开发方式。

### 项目3

切换到 Spring Boot，形成可运行版本，保留登录、系统基础入口和知识管理模块。

### 项目4

在项目3基础上增加补贴政策、信贷信息、农事服务三个业务模块。

### 项目5

整合完整 Spring Boot 后端和 Vue 前端，实现完整后台系统。
